Install

The extension is one folder of plain JavaScript — no build step. Manifest V3, works on every modern browser that ships extensions.

Chrome / Edge / Brave / Opera / Arc

  1. Download the latest .zip and unzip it.
  2. Open chrome://extensions (or edge://extensions, etc.).
  3. Toggle Developer mode on.
  4. Click Load unpacked and pick the ViperLink folder.

Firefox

  1. Visit about:debugging#/runtime/this-firefox.
  2. Click Load Temporary Add-on…
  3. Select the manifest.json file inside the unzipped folder.

Temporary add-ons unload when Firefox restarts. Permanent install requires AMO signing.

Safari

Use Apple's safari-web-extension-converter (Xcode tool) to convert the folder. The source is standard MV3 and converts without code changes.

How it works

A single content script runs on *.schoology.com. It walks visible text nodes and replaces matched URLs (https://…, http://…, bare www.…) with real anchor tags. A MutationObserver picks up content that Schoology loads via AJAX, so feed posts, profile drawers, and modal dialogs all linkify as they appear.

What it skips

  • Existing links, buttons, form fields, <code>, <pre>, and SVG.
  • Anything contenteditable — including TinyMCE, CKEditor, Froala, Quill, CodeMirror, and Monaco — so it never interferes while you're typing.
  • Elements explicitly tagged data-viperlink-skip.

What it requires

Nothing. No permissions beyond Schoology page access, no network calls, no storage, no telemetry. The whole extension is one content script that mutates the DOM.
